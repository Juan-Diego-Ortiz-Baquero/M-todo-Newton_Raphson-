Descripción del método

El método de Newton-Raphson es un algoritmo iterativo de convergencia cuadrática para encontrar raíces de funciones diferenciables. Partiendo de un valor inicial x₀, cada iteración aplica la fórmula iterativa usando la recta tangente a la curva para aproximar la raíz. Converge cuando el error absoluto entre iteraciones sucesivas es inferior a la tolerancia especificada.
